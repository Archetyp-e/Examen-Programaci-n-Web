# Proyecto Flask - Evaluación

Este proyecto corresponde al examen final de la asignatura, donde se solicita crear una aplicación web con **Flask** que incluya un menú principal 
y dos ejercicios con formularios.  

El proyecto está organizado con las carpetas `templates/` y `static/`, además de un archivo `main.py` que contiene la lógica en Python.

---

## 📂 Estructura del proyecto
- `main.py` → archivo principal con las rutas Flask.
- `templates/` → contiene las páginas HTML (`index.html`, `ejercicio1.html`, `ejercicio2.html`).
- `static/` → contiene el archivo `style.css` con los estilos.
- `requirements.txt` → dependencias necesarias para ejecutar el proyecto.
